# next-build-form-data
> Build formData by object

## installation
```bash
npm install -S afeiship/next-build-form-data --registry=https://registry.npm.taobao.org
```

## usage
```js
import nxBuildFormData from 'next-build-form-data';

// key values
nxBuildFormData({
  k1: 'image',
  k2: 'v2'
})
```
